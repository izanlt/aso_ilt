1. Normalizar a mayúsculas

Entrada: $sys = "Windows Server"
Salida: WINDOWS SERVER

```powershell
$sys.ToUpper()
```

2. Normalizar a minúsculas

Entrada: $user = "ADMINISTRATOR"
Salida: administrator

```powershell
$sys.ToLower()
```

3. Limpieza de espacios extremos

