Ejercicio 1: Menú de operaciones matemáticas

Crea un script que presente un menú con las operaciones básicas (suma, resta, multiplicación, división) y solicite al usuario que elija una operación. El script debe ejecutar la operación seleccionada utilizando case.

```bash
#!/bin/bash


echo "1) Suma"
echo "2) Resta"
echo "3) Multiplicación"
echo "4) División"
echo "Elige una opción [1-4]: "
read opcion

echo -n "Introduce el primer número: "
read num1
echo -n "Introduce el segundo número: "
read num2

case $opcion in
    1)
        resultado=$((num1 + num2))
        echo "Resultado de la suma: $resultado"
        ;;
    2)
        resultado=$((num1 - num2))
        echo "Resultado de la resta: $resultado"
        ;;
    3)
        resultado=$((num1 * num2))
        echo "Resultado de la multiplicación: $resultado"
        ;;
    4)
            resultado=$(($num1 / $num2))
            echo "Resultado de la división: $resultado"
        fi
        ;;
esac
```

Ejercicio 2: Identificación de extensión de archivo

Haz un script que solicite al usuario un nombre de archivo y, usando case, determine si es un archivo de texto (.txt), un archivo comprimido (.zip o .tar.gz), o cualquier otro tipo de archivo.

```bash
#!/bin/bash

echo -n "Introduce el nombre del archivo: "
read archivo

case "$archivo" in
    *.txt)
        echo "Es un archivo de texto (.txt)"
        ;;
    *.zip)
        echo "Es un archivo comprimido ZIP (.zip)"
        ;;
    *.tar.gz)
        echo "Es un archivo comprimido TAR GZ (.tar.gz)"
        ;;
esac

```


Ejercicio 3: Conversor de unidades

Crea un script que ofrezca un menú para convertir entre diferentes unidades de longitud (metros a kilómetros, pies a metros, etc.) utilizando case para gestionar las conversiones.

```bash

#!/bin/bash

echo "1) Metros a kilómetros"
echo "2) Kilómetros a metros"
echo "3) Pies a metros"
echo "4) Metros a pies"
echo "Elige una opción "
read opcion

echo "Introduce el valor a convertir: "
read valor

case $opcion in
    1)
        resultado=$( $valor / 1000)
        echo "$valor metros = $resultado kilómetros"
        ;;
    2)
        resultado=$($valor * 1000)
        echo "$valor kilómetros = $resultado metros"
        ;;
    3)
        resultado=$($valor * 0.3048)
        echo "$valor pies = $resultado metros"
        ;;
    4)
        resultado=$($valor / 0.3048)
        echo "$valor metros = $resultado pies"
        ;;
esac
```
